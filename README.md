# 🍜 China Beijing - Menú Digital

Aplicación web desarrollada con Astro que permite visualizar de forma dinámica el catálogo de productos de un restaurante de comida china.

El sistema consume un archivo JSON estructurado para renderizar automáticamente categorías, productos, precios por porciones e imágenes, ofreciendo una experiencia moderna, organizada y escalable.

## 🚀 Características

* 📂 Consumo de datos desde JSON (sin base de datos)
* 🧩 Renderizado dinámico de categorías y productos
* 📱 Diseño responsive tipo app de comida
* 🪄 Accordion interactivo para categorías
* 🖼️ Soporte para imágenes dinámicas por producto
* 💰 Manejo de precios por número de personas
* ⚡ Alto rendimiento gracias a Astro

## 🛠️ Tecnologías

* Astro
* TypeScript
* HTML + CSS moderno
* JSON como fuente de datos

## 🎯 Objetivo

Este proyecto fue desarrollado como práctica para fortalecer habilidades en desarrollo frontend, consumo de datos estructurados y creación de interfaces modernas enfocadas en experiencia de usuario.
